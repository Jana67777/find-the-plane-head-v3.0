🚀 寻机头小游戏 v3.0 升级公告
欢迎来到全新的 v3.0 版本！本次更新我们不仅优化了核心体验，还引入了多项重磅社交与记录功能，让您的“扫雷”之旅更加丰富有趣。

 ✨ 全新核心功能
1. 📺 全屏互动弹幕系统
- 实时互动 ：无论是经典模式还是自定义模式都支持发送弹幕，与其他玩家跨越时空“交流心得”。
- 记录上云 ：您发送的弹幕会被保存在云端数据库，并在其他玩家的游戏中随机概率展示。

2. 📼 个人对局与复盘功能
- 数据云同步 ：每一次游戏的详细步骤（点击次数、耗时、模式）现在都会自动同步到云端。只要使用同一昵称登录，即可在任何设备上查看历史战绩。
- 沉浸式复盘 ：点击任意历史记录即可进入复盘模式，支持“上一步”、“下一步”和进度条拖拽，支持导出视频完美重现您当时的【神级推理】过程。

3. 🚩 扫雷式的“标记模式”
- 在游戏控制区新增了“点击/标记”切换按钮。
- 切换至“标记模式”后，点击任意方块将直接插上红旗（或取消），不触发翻开操作，辅助推理过程。
- 电脑端支持右键标记，手机端需要点击进入标记状态。

 🛠 体验优化与调整
- 🏆 排行榜去重优化 ：排行榜现在更加清爽，同一用户仅展示其【历史最佳成绩】，让更多玩家有机会上榜。
- 🎉 游戏结束动画 ：游戏结束后（无论输赢），会自动触发全屏方块翻开的炫酷动画，让结局更具仪式感。
- 📖 新手引导增强 ：首次登录成功后，系统会自动弹出游戏规则说明，帮助新玩家零门槛快速上手。
